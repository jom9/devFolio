Test Taking and Grading Website

This project looks to help professors of the NJIT introductory Python course CS100 create and grade tests. The student takes the exam and the auto-grader checks for common errors like failing to return. Professors review the results and get to add comments. You can try it out here if you so desire. The frontend was done in vanilla Javascript, with backend using Python, PHP and MySQL.
